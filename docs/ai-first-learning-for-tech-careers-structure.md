# AI-First Learning for Tech Careers

## Purpose

This document defines the structure-first rollout for the AI-first course tracked by issue #9.

The course scaffold exists in `app/data/courses/ai-first-learning-for-tech-careers/index.ts` as a draft-only course. It is intentionally hidden from learner-facing listings and routes until lesson content is implemented and reviewed.

## Core Identity

This course teaches a system for learning anything using AI through problem-solving.

## Teaching Philosophy

- Assignment-first, AI-guided learning
- AI as a thinking partner, not an answer machine
- Reflection and knowledge extraction after every lesson
- Practice-driven progression across all modules

## Standard Lesson Template

Every lesson should follow this structure:

1. Assignment (Problem First)
2. AI Interaction
3. Guided Discovery
4. Knowledge Extraction
5. Reflection

## Skill-Wanderer Learning Cycle

1. Start with a problem
2. Use AI to explore and attempt
3. Struggle and ask better questions
4. Extract the underlying knowledge
5. Reflect and generalize
